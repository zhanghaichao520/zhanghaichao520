## 玩法说明

每个空格都是一个带图片的超链接。点击后会统计点击次数，并跳转回我的 GitHub 主页。  
脚本每次运行时，会在所有可落子位置里选择“新增点击最多”的那一步。  
GitHub Actions 通过定时任务每小时运行一次脚本。  
其余规则与普通井字棋完全一致。
